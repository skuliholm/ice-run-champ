# Icelandic Running Championships (IRC) — V1

## Core Concepts

- Championship standings are based primarily on placing and race tier.
- Elo ratings are separate power rankings.
- Hlaupadagskrá is the primary race-calendar metadata source.
- Timataka is the primary race-results ingestion source.
- Corsa support comes later.
- Stack: Next.js + Tailwind + Supabase + Vercel.

## V1 Goal

Build a publicly deployable MVP with:
- one real imported Timataka race
- realistic generated mock races
- championship standings
- Elo power rankings
- athlete/race pages
